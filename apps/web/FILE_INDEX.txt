HOW TO USE THESE FILES
================================================================================

For Full Editor Integration:
  → Use the main page: src/app/(editor)/editor/[questId]/page.tsx
  → It automatically integrates all components
  → Just pass questId as URL parameter

For Component Integration:
  1. CodeEditor
     → import CodeEditor from "@/components/editor/CodeEditor"
     → Use with ref for instance access

  2. GamePreview
     → import GamePreview from "@/components/editor/GamePreview"
     → Pass code and questId props
     → Handle onExecutionResult callback

  3. ConsolePanel
     → import ConsolePanel from "@/components/editor/ConsolePanel"
     → Pass logs and onClear callback

For Event Capture:
  → Use useCodeCapture hook
  → It automatically listens to Monaco changes
  → Returns eventCount, currentCode, takeSnapshot()

For Code Snapshots:
  → Use SnapshotManager class from src/lib/code-snapshot.ts
  → Or use takeSnapshot() from useCodeCapture hook

For OT Conversion:
  → Import functions from src/lib/ot.ts
  → Use monacoChangeToOTDelta() for delta generation
  → Use createCodeChangeEvent() for full events
